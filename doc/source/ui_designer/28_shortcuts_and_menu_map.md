# 快捷键与菜单速查

这一篇适合放在旁边随查，不讲原理，只讲入口。

## File 菜单

### 工程相关

- `Ctrl+N`：New Project
- `Ctrl+O`：Open Project...
- `Ctrl+Shift+O`：Open Example...
- `Ctrl+S`：Save Project
- `Ctrl+Shift+S`：Save As...
- `Ctrl+Shift+R`：Reload Project From Disk
- `Ctrl+W`：Close Project
- `Ctrl+E`：Export C Code...
- `Ctrl+Q`：Quit

## Edit 菜单

- `Ctrl+Z`：Undo
- `Ctrl+Shift+Z`：Redo
- `Ctrl+A`：Select All
- `Ctrl+C`：Copy
- `Ctrl+X`：Cut
- `Ctrl+V`：Paste
- `Ctrl+D`：Duplicate
- `Del`：Delete

## Build 菜单

- `F5`：Build EXE && Run
- `Ctrl+F5`：Rebuild EGUI Project

高频菜单项：

- `Auto Compile`
- `Stop Exe`
- `Release Build (EXE)...`
- `Release Profiles...`
- `Release History...`
- `Repository Health...`
- `Generate Resources`

## View 菜单

### 预览布局

- `Ctrl+1`：Vertical
- `Ctrl+2`：Horizontal
- `Ctrl+3`：Overlay Only
- `Ctrl+4`：Swap Preview/Overlay

### 缩放

- `Ctrl+=`：Zoom In
- `Ctrl+-`：Zoom Out
- `Ctrl+0`：Zoom Reset (100%)

### 其他高频入口

- `Workspace -> Project / Structure / Components / Assets`
- `Inspector -> Properties / Animations / Page`
- `Tools -> Diagnostics / History / Debug Output`
- `Focus Canvas`
- `Show Grid`

## 最值得记住的一组快捷键

如果你不想记太多，先记这 8 个就够用了：

- `Ctrl+Shift+O`
- `Ctrl+S`
- `Ctrl+Z`
- `Ctrl+C`
- `Ctrl+V`
- `F5`
- `Ctrl+F5`
- `Ctrl+0`

## 菜单记忆方法

可以按用途记：

- `File`：工程进出
- `Edit`：选中对象怎么改
- `Arrange`：几何和层级
- `Build`：生成和发布
- `View`：界面怎么显示
