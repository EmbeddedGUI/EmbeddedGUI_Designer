Bundled Designer examples live here.

- `DesignerSandbox`: a lightweight playground for day-to-day Designer checks.
- `HelloSimpleDemo`: a minimal hello-style EGUI project for demo and onboarding.
- `HelloWidgetFallbackDemo`: demonstrates app-local custom widget fallback when `egui_view_*.h` parsing is insufficient and a manual `custom_widgets/*.py` descriptor is provided.
- `ResourceGeneratorDemo`: a standalone Resource Generator playground for the common single-text font workflow.
- `ResourceGeneratorMultiTextDemo`: a standalone Resource Generator playground for fonts that merge multiple `txt` files into one output.
