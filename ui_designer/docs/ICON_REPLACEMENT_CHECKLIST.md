## 6. 常见错误（避免返工）

- 同一语义在不同页面使用不同图标。
- 仅替换 default 态，遗漏 hover/active/disabled。
- 图标尺寸正确但点击热区过小。
- 直接写死 `#xxxxxx` 而不是使用 token。
- 图标视觉居中但未做光学对齐，导致“看起来偏”。
